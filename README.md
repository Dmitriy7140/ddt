# Прототип системы детекции уборки столиков

## Описание

Этот проект — упрощенный прототип системы, которая фиксирует события за одним выбранным столиком в кафе/ресторанe по видео.  

Цель: показать основной пайплайн обработки видео, фиксацию событий и простую аналитику.

---

## Функционал

Система фиксирует три состояния столика:

1. **EMPTY** — стол пуст (людей в зоне нет)  
2. **OCCUPIED** — стол занят (есть движение/человек в зоне)  
3. **APPROACH** — появление человека за столиком после периода пустоты  

Дополнительно фиксируется задержка между уходом предыдущего гостя и подходом следующего.

---

## Выбранный столик

Столик выбирается вручную через интерфейс OpenCV (`cv2.selectROI`) при запуске скрипта.  
На видео рисуется bounding box, цвет которого зависит от состояния:  
- Зеленый — пусто  
- Красный — занят

---

## Технологии

- **Python 3.8+**  
- **OpenCV** — работа с видео, выделение ROI, вычитание фона, отрисовка  
- **Pandas** — хранение событий и расчет статистики  
- (опционально) **Ultralytics YOLO** для детекции людей
